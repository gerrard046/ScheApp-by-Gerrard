# ScheApp Pro: Titan Edition ☀️🌙🚀

ScheApp Pro adalah platform manajemen jadwal elite yang dirancang untuk memaksimalkan produktivitas dengan estetika yang memukau. Edisi **Titan** membawa fitur-fitur tingkat lanjut untuk individu maupun kolaborasi grup.

## 🌟 Fitur Utama (Titan Expansion)

### 1. 📝 Sub-Tasks & Checklist
Jangan biarkan tugas besar menghambatmu. Pecah setiap agenda menjadi sub-tugas kecil yang bisa dicentang, lengkap dengan progress bar otomatis.

### 2. 📸 Proof of Completion (Real File Uploads)
Validasi hasil kerjamu dengan bukti nyata.
- **Upload Langsung**: Pilih file (PDF, Gambar, Video) langsung dari komputermu.
- **Auto-Detect**: Sistem otomatis mengenali jenis file dan menampilkan ikon yang sesuai.
- **Verifikasi Admin**: Admin dapat meninjau bukti foto sebelum memberikan bonus XP.

### 3. 🌙 Midnight Sunset (Dark Mode)
Bekerja dengan nyaman di segala kondisi pencahayaan. Ganti antara mode **Sunset Sunshine** (Terang) dan **Midnight Sunset** (Gelap) dengan satu sentuhan di Navbar.

### 4. 🍅 Focus Mode & Lo-Fi Pomodoro
Masuk ke zona fokus tanpa gangguan. Gunakan timer Pomodoro terintegrasi sambil mendengarkan musik **Lo-Fi Beats** (Chill Hop & Jazz).

### 5. 📄 Productivity Snapshot (PDF Reports)
Pantau grafik produktivitasmu. Cetak laporan mingguan yang rapi dan profesional langsung ke format PDF.

### 6. 🔔 Smart Notification Center
Dapatkan update real-time. Notifikasi akan muncul saat tugas diverifikasi admin, ada tugas grup baru, atau pengumuman penting lainnya.

### 7. 🎮 Gamification Level Elite
Dapatkan XP untuk setiap tugas yang diselesaikan. Capai level tertinggi dan jadilah yang paling produktif di Leaderboard!

## 🚀 Cara Menjalankan

1. **Clone & Install**:
   ```bash
   git clone https://github.com/gerrard046/ScheApp-by-Gerrard.git
   composer install
   npm install
   ```

2. **Environment Setup**:
   Salin `.env.example` ke `.env` dan atur database (SQLite).
   ```bash
   php artisan key:generate
   php artisan migrate
   php artisan storage:link
   ```

3. **Run App**:
   ```bash
   php artisan serve
   ```
