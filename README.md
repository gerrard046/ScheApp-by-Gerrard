## ğŸ“– Deskripsi
**ScheApp** adalah aplikasi manajemen jadwal berbasis web yang dirancang untuk membantu pengguna mengatur agenda harian secara efisien. Proyek ini dibangun dengan fokus pada kemudahan penggunaan dan performa yang ringan.

### Tujuan Utama:
* Menyediakan platform pencatatan jadwal yang intuitif.
* Mengatur prioritas tugas agar pengguna tetap produktif.
* Memberikan notifikasi jadwal secara tepat waktu.
